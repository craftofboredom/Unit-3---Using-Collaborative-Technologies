Macros

I can select an appropriate combination of IT tools and devices to carry out collaborative tasks

I’ve noted down a list of relevant hardware and software to have a successful collaborative meeting. This is my current setup that I use myself for personal, gaming and for this course. This example is what I use to successfully and productively join our daily classes for this course, art and gaming.

User will need the following hardware

  • Creative webcam with built-in microphone
  • Creative Bluetooth Pebble speakers
  • MSI Optix G27 Series monitor (27 inch, 2560 x 1440 pixels native)
  • MSI Laptop (Intel i7, 16GB RAM, 125GB SSD, 1TB HD, 5TB External HD)
  • 150 Mbps minimum broadband speed (router by broadband provider)
  • KLIM Wireless keyboard
  • Steelseries Rival 3 Wireless mouse
  • Smart phone

Software

Description

Use

Windows 10 Main OS This is the main operating system I use for this course, art and gaming.
Google Drive Cloud or content management system All my important files are backed up in Google drive. This includes all my coursework as well. If file sharing is allowed, other Google users are also able to access my files and make relevant amendments. This is also where I save all my scanned and digital art.
Github website and Desktop Online content management system Online site where all my coursework is saved and uploaded. This is linked with Google drive as well.

Google Chrome Web browser which also includes Emails, Docs, Sheets and Slides Web based system where I access my emails which is used to communicate with my tutor. My emails can send my coursework as long as it’s no larger than 25MB. If the file is larger, this can be shared via Google Drive instead.
WhatsApp Desktop and Mobile Instant messaging service that is synced with my phone. Allows the creation of a group of people who are in my course. I don’t use it for this course, but I do use it for my job and friends.
Adobe Connect Online meeting / classroom environment Connects myself with my tutor and other learners to learn about Software Development.
Discord Allows audio communication between a group of people Connects myself via chat and audio with other gamers when I’m playing online.
Deviantart Social network for artists Allows me to upload my art online with other like-minded people. It allows me to like, comment and follow other artists. It’s quite similar to Facebook and Instagram.
